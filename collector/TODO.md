# 설계

## 수집기 기능
* 원클릭 수집
    * 형태 (캔들 or 주문)
    * 거래소
    * 시간 스케일
    * 기간 (언제까지?)
* 다중 수집
    * 멀티 스레딩 or 프로세싱
* DB 현황 확인 기능
    * 테이블 삭제 및 쿼리 UI
* 시각화 기능
